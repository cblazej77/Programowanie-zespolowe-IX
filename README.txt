Jak rozpocząć pracę nad aplikacją?
Rozpoczać od kroków podanych w poradnikach:
https://reactnative.dev/docs/environment-setup
https://docs.expo.dev/get-started/installation/
Następnie doinstalować w expo pakiety:
formik styled-components expo-constants @react-native-community/datetimepicker
Najlepiej za pomocą komendy: npx expo install formik styled-components expo-constants @react-native-community/datetimepicker
Można rozpoczynać pracę :)
